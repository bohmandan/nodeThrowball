# nodeThrowball
=============

nodeThrowBall - A demo multiplayer game in HTML5, using socket.io, node.js and canvas.

## Built by:
Generated skeleton with express-generator

cmd: 
```shell
express nodeThrowball
```

### installed with:
npm install

### Set debug with:
SET DEBUG=nodeThrowball:*

### Run with:
npm start
